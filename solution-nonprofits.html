FAQ

Questions from non-profit teams.

Can this work with our existing donation platform?

Yes. We integrate with your current stack where possible, then add automation around it so you avoid disruptive migrations.

Do you support recurring campaigns and Ramadan peaks?

Yes. We design separate automation paths for recurring campaigns and seasonal surges to keep response times stable.

Can staff members still override automations manually?

Absolutely. Every workflow includes manual override points so teams keep control when edge cases appear.

Can we keep some workflows manual while automating others?

Yes. We implement automation in phases and keep manual checkpoints wherever your team needs control.

Will this help with trustee and board reporting?

Yes. We create standardized reporting views so governance updates are generated consistently each cycle.
